# 🚨 AI Incident Intelligence Dashboard

A live AI-powered dashboard to monitor incidents, classify risk, 
and generate automated reports for ops and delivery teams.

## 🎯 Problem It Solves
Ops teams dealing with high incident volumes struggle to prioritise 
what needs immediate attention. This system automates risk classification 
and surfaces critical issues instantly.

## ✨ Features
- 📊 Key Metrics — Total, Open, and High Severity incidents at a glance
- 🚨 High Risk Incidents panel with automatic risk scoring
- 📋 Full incident data table with colour-coded severity
- 🔍 Dynamic filters by Service (Azure Storage, API Gateway, Database etc.)
- 🔍 Filter by Severity (High, Medium, Low)
- 🤖 One-click AI Incident Report generation

## 🧠 Tech Stack
`Python` `Streamlit` `Pandas` `OpenPyXL`
